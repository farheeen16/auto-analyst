# Auto-Analyst: AI Research Assistant

Auto-Analyst is an end-to-end Retrieval-Augmented Generation (RAG) system that allows users to upload research papers (PDFs), build a knowledge base, and ask research questions with citations, intent routing, and evaluation metrics.
